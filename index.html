Character classes

At start of the game, you may choose a class from following

  • Barbarian Strong melee build, uses brute force difficulty: easy
  • Alchemist agile, better loot, unique items difficulty: medium
  • Wizard weak, fun spells, first strike difficulty: easy

Enemy types

Learn about different enemies

  • a knight Strong melee, lots of armor, slow, weak to poison difficulty: easy
  • an undead priest weak, high damage, agile, weak to holy difficulty: medium
  • a small dragon Strong, multiple attacks, final boss, weak to fire difficulty: hard

Items

Different items and how to use them

  • weapons: type "attack" or type the weapon's name to use it.
  • potions: type the name of the item to use it. These are single time uses.
  • magic: type "magic" and then the spell's index to use it. Have limited uses.
